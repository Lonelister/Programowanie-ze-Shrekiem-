Podstawowe właściwości CSS

Cascading Style Sheets (CSS) to język służący do stylizacji stron internetowych. Pozwala na kontrolowanie wyglądu elementów na stronie. Oto kilka podstawowych właściwości CSS:

color

Kontroluje kolor tekstu.

font-family

Określa rodzaj czcionki.

font-size

Ustala rozmiar czcionki.

Ten tekst jest większy.

text-align

Określa wyjustowanie tekstu (np. lewo, prawo, środek).

Ten tekst jest wyjustowany do prawej.

background-color

Definiuje kolor tła elementu.

Ten tekst ma tło w kolorze niebieskim.

border

Tworzy obramowanie wokół elementu.

Ten element ma czerwone obramowanie.

margin

Kontroluje odstęp między elementami.

Ten element ma zielone obramowanie i odstęp 20px od innych elementów.

padding

Określa odstęp między zawartością a obramowaniem elementu.

Ten element ma niebieskie obramowanie i wewnętrzny odstęp 20px.

width

Określa szerokość elementu.

Ten element ma szerokość 200px i tło w kolorze żółtym.

height

Ustala wysokość elementu.

Ten element ma wysokość 100px i tło w kolorze zielonym.

display

Kontroluje sposób wyświetlania elementu (np. block, inline, flex).

Ten element jest wyświetlany jako blok.

position

Określa sposób pozycjonowania elementu (np. relative, absolute, fixed).

Ten element jest pozycjonowany względem swojego pierwotnego miejsca.

float

Pozwala elementom "unikać" innych i ustala sposób ich rozmieszczenia na stronie.

Ten element jest wyrównany do lewej strony.

Ten element jest wyrównany do prawej strony.
